#gerar imagem FAT
sudo apt-get install dosfstools
mkfs.fat -F 32 -n LABEL -S 512 -C discoFAT.img 8192
-F 32: Especifica que você deseja criar um sistema de arquivos FAT32. Você pode ajustar isso para FAT12 ou FAT16, dependendo das suas necessidades.
-n LABEL: Define um rótulo para o sistema de arquivos. Substitua "LABEL" pelo rótulo desejado.
-S 512: Especifica o tamanho do setor. Geralmente, os sistemas de arquivos FAT usam setores de 512 bytes.
-C imagem_fat.img: Especifica o nome do arquivo de imagem de disco a ser criado.
8192: Especifica o número de setores na imagem. Você pode ajustar isso conforme necessário.


#montar imagem de disco FAT
sudo mount -o loop discoFAT.img /FAT


#desmontar imagem de disco FAT
sudo umount /FAT
